💡 Cheat-sheet (Çit-list)

Bu səhifə testçi və proqramçılar üçün ən çox istifadə olunan məlumatları tez tapmaq üçün "əl altında köməkçi qeyd dəftəri" funksiyasını yerinə yetirir.

✅ Ümumi yoxlamalar (Veb və Mobil)

  • 🔗 Linklər işləyirmi?
  • 📋 Formadakı sahələr düzgün çıxırmı?
  • 📱 Responsiv dizayn düzgündürmü?
  • ⚠️ Xəta mesajları düzgün göstərilirmi?

🧪 Veb formaları test etmək üçün

  • Boş buraxılmış sahələr üçün yoxlama
  • Səhv formatda məlumat daxil edilməsi (məs. email)
  • Maksimum/minimum uzunluq
  • Xüsusi simvolların təsiri

🌐 Lokallaşdırma yoxlamaları

  • Dil tərcümələri düzgünmü?
  • Tarix və rəqəmlər ölkəyə uyğun göstərilirmi?
  • RTL/LTR dizayn dəstəyi

💻 Proqramçılar üçün – Git

  • git status — dəyişiklikləri göstərir
  • git add . — bütün faylları əlavə edir
  • git commit -m "mesaj" — kommit edir
  • git push — dəyişikliyi göndərir

🖥 Bash (Terminal) komandaları

  • ls — qovluqdakı faylları siyahıla
  • cd — qovluq dəyişdir
  • mkdir — yeni qovluq yarat
  • rm — faylı sil

🗃 SQL əsas əmrlər

  • SELECT * FROM users; — bütün məlumatları seç
  • INSERT INTO users (name) VALUES ('Ali');
  • UPDATE users SET name='Veli' WHERE id=1;
  • DELETE FROM users WHERE id=2;
