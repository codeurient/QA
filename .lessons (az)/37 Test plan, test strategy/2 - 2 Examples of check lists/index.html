✅ High-level Test Case Nümunəsi

Test Case ID:
TC-HIGH-001
Başlıq:
İstifadəçinin uğurla qeydiyyatdan keçməsi
Təsvir:
Bu test halı istifadəçinin sistemdə uğurla qeydiyyatdan keçib-keçmədiyini ümumi şəkildə yoxlayır.
Əvvəlcədən tələb olunan şərtlər:
Heç biri. İstifadəçi sistemdə qeydiyyatdan keçməyib.
Test Addımları:
- İstifadəçi qeydiyyat səhifəsini açır
- Lazımi sahələri doldurur
- Qeydiyyat düyməsinə klikləyir
Gözlənilən nəticə:
İstifadəçiyə "Uğurla qeydiyyatdan keçdiniz" mesajı göstərilməlidir və sistemə daxil edilməlidir.
